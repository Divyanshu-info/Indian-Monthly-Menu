# UI-Patterns: Pattern Reference

> Detailed descriptions of proven UI patterns.

## Feature Showcase with Tabs

**When:** SaaS landing pages with multiple core features.
**What:** Horizontal tab bar with alternating content below (image + text). Each tab showcases one feature.
**Reference:** Dribbble designs, Behance projects -- dominant SaaS pattern.

## Sticky Section with Scroll Swap

**When:** Product explanations with multiple aspects.
**What:** Left column remains sticky, right column scrolls and swaps content.
**Reference:** Apple-inspired, widely seen on Godly and Awwwards.

## Marquee Social Proof

**When:** B2B SaaS, platforms with well-known customers.
**What:** Horizontally auto-scrolling logo row. Infinite loop with no visible start/end.
**Implementation:** CSS-only with `animation: translateX` or Framer Motion.

## Command Palette

**When:** Power user tools, developer products, dashboards.
**What:** Cmd+K opens a search field with fuzzy matching.
**Reference:** Reflect, Linear, Notion (all Godly-featured).

## Mega-Footer as Conversion Zone

**When:** SaaS products, last conversion opportunity before users leave.
**What:** Footer area begins with a large CTA block ("Ready to get started?"), followed by a classic footer with links.

## Comparison Table

**When:** Products differentiating from competitors.
**What:** Column comparison with checkmarks/crosses. Own product highlighted.
**Reference:** Lapa Ninja SaaS landing pages.

## Onboarding Wizard

**When:** Complex products with a setup process.
**What:** Multi-step guided process with progress bar. Maximum 5 steps.
**Reference:** Dribbble dashboard designs.
