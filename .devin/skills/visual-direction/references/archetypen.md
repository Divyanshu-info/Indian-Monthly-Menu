## Typography Trends (2026)

### 1. Editorial Serif Renaissance
Display serifs are making a comeback. Playfair Display, Fraunces, Newsreader in hero sections. Premium effect.

### 2. Mono as Design Statement
JetBrains Mono, Geist Mono, Berkeley Mono used not only for code but as a design element in CTAs and labels. Reference: Vercel, Linear.

### 3. Ultra-Condensed Headlines
Tight tracking at large sizes. Aeonik Condensed, Instrument Sans Condensed. CSS Winner SOTD winners showcase this trend.

### 4. Handwriting Accents
Occasional handwritten elements as contrast to clean typography. Individual words or annotations, not entire blocks.

## Visual Trends (2026)

### 1. Gradient Mesh as Background
Soft, multi-color gradients with a mesh effect. More subtle than flat gradients.

### 2. Glassmorphism 2.0
More subtle effects: `backdrop-filter: blur(8-16px)` with low opacity. As a depth layer, not heavy-handed.

### 3. Grain/Noise Textures
Fine noise (2-5% opacity) on backgrounds for tactile quality. Especially in dark-theme designs.

### 4. Duotone Photography
Two-color image treatment for consistent visual language. Strong branding effect.

### 5. Abstract Blobs
Organic, amorphous shapes as background decoration. Softer than geometric shapes.

## Layout Archetypes (2026)

### Bento Grid
Unevenly sized cards in a grid formation. Large card left, smaller cards right. Vercel Ship, Notion, Apple.

### Stacked Sections
Full width, vertically stacked, each section with its own color scheme. Clear separation through color/spacing.

### Split Panel
50/50 or 60/40 horizontal division. Image + text or two content blocks.

### Masonry Flow
Pinterest-style layout with variable heights. For galleries and portfolio grids.
